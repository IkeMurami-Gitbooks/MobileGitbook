# Установка

```
pip3 install frida
pip3 install frida-tools
```

Frida-сервер — нужно поднимать на стороне мобильного приложения

* iOS: [cydia](https://frida.re/docs/ios/)
* Android: [frida-server-\*-android-\*](https://github.com/frida/frida/releases)
